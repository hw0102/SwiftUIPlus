# Code of Conduct

## Our Standard

This project expects contributors to participate in a respectful, constructive, and professional way.

Examples of behavior that contribute to a positive environment:

- being respectful in disagreements
- giving actionable and technical feedback
- assuming good intent while still being direct
- accepting responsibility for mistakes and correcting them
- focusing discussion on the work rather than the person

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- insulting or dismissive language
- trolling or deliberately disruptive behavior
- publishing private information without permission
- conduct that would make collaboration unsafe or unnecessarily hostile

## Scope

This Code of Conduct applies in:

- GitHub issues
- pull requests
- discussions related to this project

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this Code of Conduct.

Repeated or severe violations may result in temporary or permanent exclusion from the project community.

## Reporting

If you experience or observe behavior that violates this Code of Conduct, contact the project maintainer directly through the contact method listed on the repository profile or an appropriate private channel.

## Notes

This document is intentionally short and practical. The goal is straightforward: keep collaboration respectful, useful, and safe for contributors.
